# Patterns JS
Examples of programming patterns, written in JavaScript.

Open index.html in some sort of server (e.g. use Live Server extension for VS Code).

The list of implemented patterns:

Creational:
- Builder
- Factory Method
- Prototype
- Singleton

Structural:
- Adapter
- Bridge
- Composite
- Decorator
- Facade
- Flyweight
- Proxy

Behavioral:
- Chain Of Responsibility
- Command
- Iterator
- Mediator
- Memento
- Observer
- State
